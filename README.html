História dos computadores

(Voltar)

O 1° computador é cosiderado por muitos o MARK I, o início da Gerra Fria muito outros computadores sugiram pelo mundo assim iniciando a 1° geração de computadores. Em meados de 1959 teve início a 2° geração de computadores onde os computadores passaram a ter um tamanho reduzido, nessa é já havia a possibilidade de locomover os computadores para outras salas. Com a criação dos circuitos integrados e os chips veio a 3° geração de computadores por volta de 1965, Em 1971 a 4° geração de computadores, que é a que vivemos, veio possibilitar que cada cidadão possua seu proprio PC.

O que são Clients?

(Voltar)

São dados distribuidos através de um server via navegador ou aplicativo em um equipamento eletronico como celular, PC, televisão smart, entre outros.

O que são Servers?

(Voltar)

É uma máquina que armazena informações de dados de arquivos. Ela é acessada via URL. Você digita o que busca no seu navegador , exemplo, foto de carro da Porsche, com isso o servidor localiza um foto em seus dados e transmite para a tela do client no seu equipamento eletronico. Introdução ao HTML

(Voltar)

Para utilizar o HTML precisamos de um navegador, uma ferramenta para digitar os códigos fontes como exemplo Visual Studio Code. Existe uma ferramenta facilitadora que é o Inspetor de Elementos ela, no Windows é acessada utilizando a tecla F12, ao acionala você tem acesso aos códigos usados para criar o site em que você esteja visitando. Os HTMLs são divididos em 3 partes. Parte 1 HTML inicia e finaliza todos os elementos que nele constam. Parte 2 Chamada de Head, essa parte é responsável pelo que aparece no nome da guia do navegador. Parte 3 A parte com mais conteúdo de costume, o famoso Body, o corpo do site, todas as interações, imagens e texto que compõem um site estão dentro dessa parte.

HTML é a sigla para Linguagem de Marcação de Hipertexto, e pode ser definida como uma linguagem de computador – mas não de programação, e veremos o porquê em breve. Ela é amplamente utilizada para desenvolver páginas da web, uma vez que documentos HTML são facilmente interpretados pelos navegadores (Google Chrome, Mozilla Firefox, Opera, Safari etc). O HTML nasceu da junção entre os padrões HyTime e SGML. O HyTime é um padrão que trabalha com a representação estruturada de hipermídia e outros conteúdos que dependem de tempo, como áudio e vídeo, por exemplo. Awari Code
